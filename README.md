# nbp-api

## Clone repository

```
git clone https://github.com/jedrzej-smok/nbp-api.git
```

## Create virtual environment, activate it and install required packages

```
cd nbp-api
python -m venv ./venv
.\venv\Scripts\Activate.ps1
pip install -r requirements.txt
```

## Run application
```
python src/main.py
```

## Test code
```
pytest
```
